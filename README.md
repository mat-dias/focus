# focus
Focus - produtividade pessoal

OBJETIVO:

Desenvolver um sistema web de produtividade pessoal com foco na aplicação da técnica Pomodoro, auxiliando o usuário a melhorar sua concentração, organização e desempenho nas tarefas diárias.
O sistema permite controlar ciclos de foco e pausas, promovendo maior disciplina no gerenciamento do tempo e aumento da eficiência produtiva.

TECNOLOGIAS UTILIZADAS:

BackEnd: 
- PHP 
- MySQL 
- MySQLi 


FrontEnd: 
- HTML5 
- CSS3 
- JavaScript

Ambiente de Desenvolvimento: 
- Servidor local (XAMPP, WAMP ou similar) 
-Apache 
  
INSTRUÇÕES DE USO
1. Configuração do Ambiente:
- Instalar um servidor local (XAMPP, WAMP ou similar).
- Instalar e configurar o WampServer.
- Criar o banco de dados do projeto.
2. Configuração da Conexão:
- Acessar o arquivo de conexão com o banco de dados.
- Inserir as credenciais corretas (host, banco, usuário e senha).
3. Execução:
- Colocar a pasta do projeto dentro do diretório do servidor (ex:htdocs).
- Iniciar o Apache e o Wamp (Ou servidor escolhido).
- Acessar no navegador: http://localhost/Focus1.8/
4. Utilização:
- Realizar login no sistema.
- Utilizar o temporizador Pomodoro para iniciar ciclos de foco.
- Acompanhar as sessões de produtividade conforme implementado no sistema.
